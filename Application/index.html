Auto-Deploy Project

My project involves automating the deployment process
of web applications on AWS using Terraform
for infrastructure provisioning and Jenkins for CI/CD pipeline automation.
The pipeline involves pulling code from GitHub,
running tests, building Docker images, pushing them to DockerHub, and then
deploying the application onto AWS instances with Terraform code.
The infrastructure setup includes creating a VPC, internet gateway, public subnets,
EC2 instances with Docker Images, and an Application Load Balancer
with Target Groupto distribute traffic.
Finally, Bootstapping in AWS is used to configure the EC2 instances and
deploy Docker containers with the web application.
